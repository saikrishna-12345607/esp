This project is a classic Snake Game developed using ESP32, LED Matrix Display, and a Joystick module.
The snake moves based on the joystick direction . 
The goal of the game is to eat the food that appears randomly on the LED matrix.
Each time the snake eats the food, its length increases. 
The game ends if the snake hits the wall or its own body.
Components Used:
ESP32 Development Board
8x8 LED Matrix Display
Joystick Module
Connecting Wires
Breadboard
